# Paul Graham's Essays in Markdown (converted to PDF & ePub)

A Unix-friendly way (less some Python, sorry about that) of

 1. grabbing Paul Graham's essays,
 2. converting them to Markdown, and
 3. generating both
    i. a PDF, and
    ii an ePub
